# Phase 2: 미분의 본질과 연쇄 법칙 (수학 기초)

이 단계에서는 머신러닝과 역전파의 톱니바퀴를 돌리는 강력한 수학적 기초를 다집니다. 이 수학 공식들이 `microgpt.py` 내부의 `_local_grads` 에 어떻게 코드로 구현되어 있는지 연결하며 학습합니다.

## 📌 [Chapter 1. 기초 함수와 미분법 (Calculus)](02_chapter_01_02.md)
모든 딥러닝 망은 아주 단순한 함수들의 연결거리입니다. 복잡한 수식이 아니라, 오직 우리가 사용할 함수들의 기본 미분법 결과만 확실히 외우고 넘어가면 됩니다.
* **1-1. 다항식의 미분 (Power Rule)**: $f(x) = x^n$ 의 미분과 `__pow__` 메서드의 구현 원리
* **1-2. 자연로그 함수의 미분**: $f(x) = \ln(x)$ 의 미분($1/x$)과 역전파 시퀀스의 크로스 엔트로피 관련 로그 연산
* **1-3. 지수 함수의 미분 (Exponential)**: $f(x) = e^x$ 의 미분($e^x$)과 소프트맥스 처리 연결

## 📌 [Chapter 2. 다변수 함수와 편미분 (Partial Derivative)](02_chapter_01_02.md#chapter-2-다변수-함수와-편미분-partial-derivative)
우리의 함수는 $f(x) = x \times 2$ 처럼 변수가 하나만 있는 것이 아니라 두 개의 값(Value)이 서로 섞입니다. ($f = x + y$, $f = x \times y$). 이때 미분(기울기)이 나눠지는 원리를 이해합니다.
* **2-1. 편미분의 개념**: 한 변수에 대해서만 미분하고 나머지 변수를 모른 척(상수 취급) 하는 원리
* **2-2. 덧셈의 편미분**: $x + y$ 의 편미분. 왜 덧셈 노드(`__add__`)에서 만들어지는 로컬 그래디언트는 항상 `(1, 1)`인가?
* **2-3. 곱셈의 편미분**: $x \times y$ 의 편미분. 서로의 값을 교차해서 영향력을 교환하는(Cross) 곱셈 노드(`__mul__`)의 성질

## 📌 [Chapter 3. 연쇄 법칙 (Chain Rule, 핵심 엔진)](03_chapter_03.md)
역전파(Backpropagation)라는 마법을 코드로 구현 가능하게 해주는 유일한 수학 법칙입니다. 각 노드들이 자기가 아는 '로컬 그래디언트(나의 영향력)'만으로 전체 망의 미분값을 구해내는 과정을 배웁니다.
* **3-1. 합성 함수(Composite Function)의 미분**: 함수 안에 함수가 있을 때 껍질을 벗기며 곱해 들어가는 공식
* **3-2. 로컬 그래디언트와 글로벌 그래디언트의 분리**: 코딩(객체 지향)에 맞추어 연쇄 법칙을 관점적으로 분리하기
* **3-3. 미분값의 누적 (Gradient Accumulation)**: 여러 개의 가지(Branch)로 정보가 뻗어나갔을 때, 되돌아오는 미분값들을 어떻게 취합(`+=`)해야 하는가?

## 📌 [Chapter 4. 확률 분포와 손실 함수의 수학](04_chapter_04.md)
모델의 출력을 확률로 변환하고, 정답과 얼마나 다른지를 숫자 하나로 수렴시키는 수학적 장치를 배웁니다.
* **4-1. 확률 분포와 소프트맥스 공식**: 임의의 실수 벡터를 \"합이 1이 되는 양수\" 확률 벡터로 변환하는 $\frac{e^{x_i}}{\sum e^{x_j}}$ 공식
* **4-2. 크로스 엔트로피 손실 (Cross-Entropy Loss)**: 정답 토큰의 확률에 $-\ln(p)$ 를 씌워 \"정답을 얼마나 잘 맞혔는가\"를 하나의 스칼라로 측정하는 원리
* **4-3. 수치 안정성 트릭 (Max Subtraction)**: `max_val`을 빼도 소프트맥스 결과가 변하지 않는 수학적 증명과 오버플로우 방지
