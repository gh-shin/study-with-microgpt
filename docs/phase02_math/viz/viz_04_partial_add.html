덧셈 노드의 편미분: f(x, y) = x + y

x와 y의 값을 슬라이더로 움직여보세요. x가 1 증가하면 f도 똑같이 1 증가하고, y가 1 증가해도 f는 똑같이 1 증가합니다.

f(x, y) = x + y

∂f / ∂x = 1 (x를 아주 조금 변화시키면, 그만큼만 f가 변환함)
∂f / ∂y = 1 (y를 아주 조금 변화시키면, 그만큼만 f가 변환함)
💡 로컬 미분값 (1, 1): 더하기 노드(`__add__`)는 x가 크든 작든 상관없이, 위에서 흘러내려온 글로벌 미분값을 두 개의 자식 노드에게 1배수로 똑같이(공평하게) 흘려보냅니다!
