# Phase 4: Autograd Engine 실전 구현 및 분석 (최종 목표)

지금까지 배운 파이썬 객체지향 기술과 연쇄 법칙(Chain Rule) 수학을 결합하여, 현재 파일(`microgpt.py`)의 `Value` 클래스를 직접 구현하고 신경망의 구조로 확장하는 최종 단계입니다.

## 📌 [Chapter 1. 단일 스칼라 노드 객체화 (Value 클래스 뼈대)](02_chapter_01_02.md)
가장 작고 단순한 기초 블록인 `Value` 클래스의 기본 뼈대를 만듭니다.
* **1-1. 상태의 캡슐화**: `data`와 `grad`, 그리고 족보를 저장하는 생성자 구현
* **1-2. 메모리 최적화**: 수백만 개의 객체 생성을 대비한 `__slots__` 적용

## 📌 [Chapter 2. 연산 노드 생성기 구현 (매직 메서드 오버로딩)](02_chapter_01_02.md#chapter-2-연산-노드-생성기-구현-매직-메서드-오버로딩)
수학적 연산을 객체 간의 관계로 매핑합니다. 연산이 일어날 때마다 새로운 노드가 탄생하며 그래프가 자라나게 합니다.
* **2-1. 수학 기호의 객체화**: `__add__`, `__mul__` 등 사칙연산의 오버로딩과 지역 미분값(`_local_grads`) 정의
* **2-2. 파이썬 타입 방어**: `isinstance`를 활용한 상수 호환 및 `__radd__`, `__rmul__` 교환 법칙 구현

## 📌 [Chapter 3. DFS를 활용한 체인룰 계산기 (`backward`)](03_chapter_03_04.md)
정의된 미분 공식들과 방향성 그래프(DAG)를 바탕으로, 전체 오차를 파라미터로 전파하는 핵심 엔진을 조립합니다.
* **3-1. 위상 정렬과 재귀**: `build_topo` 함수를 통한 깊이 우선 탐색(DFS) 정렬 리스트 구축
* **3-2. 글로벌 그래디언트의 누적**: 리스트를 역방향으로 순회하며 체인 룰에 맞춰 자식들의 `.grad`를 누적(`+=`)하는 반복문 작성

## 📌 [Chapter 4. 신경망(NN)으로의 확장 및 검증](03_chapter_03_04.md#chapter-4-신경망nn으로의-확장-및-검증)
우리가 짠 이 1차원 스칼라(Scalar) `Value` 객체 수십, 수백 개가 뭉쳐서 어떻게 `Linear(선형 레이어)` 행렬곱이나 `Softmax` 확률 변환을 이뤄내는지 배웁니다.
* **4-1. 행렬 연산(Linear)의 구현**: 반복문과 리스트 내포(List Comprehension)를 이용해 2차원 어텐션 가중치 행렬과 `Value` 값 곱하기
* **4-2. 손실 계산과 옵티마이저 (Adam)**: 파라미터(`w`)들이 자신의 `grad` 값만큼 스스로를 깎아내려(`p.data -= lr * p.grad`) 결국 언어 모델(GPT)이 정답을 맞히게 되는 학습 과정의 이해

## 📌 [Chapter 5. 하이퍼파라미터와 파라미터 초기화](04_chapter_05.md)
모델의 뼈대(구조)를 결정하는 설정값들과, 가중치의 초기 상태를 준비하는 과정을 배웁니다.
* **5-1. 하이퍼파라미터의 의미**: `n_layer`, `n_embd`, `block_size`, `n_head`, `head_dim`의 각 역할과 상호 관계
* **5-2. 가우시안 랜덤 초기화**: 왜 가중치를 0이 아닌 작은 랜덤 값으로 시작하는가? `std=0.08`의 의미
* **5-3. `state_dict` 구조**: 모델의 전체 가중치 저장소(딕셔너리)를 조립하고 파라미터 리스트로 평탄화하는 과정

## 📌 [Chapter 6. 학습 루프와 Adam 옵티마이저](05_chapter_06.md)
모델이 실제로 지식을 습득하는 반복 학습 과정, 그리고 그래디언트를 활용해 가중치를 업데이트하는 최적화 알고리즘을 상세히 배웁니다.
* **6-1. 학습 루프의 구조**: 순전파 → 손실 계산 → 역전파 → 파라미터 갱신의 4단계 반복
* **6-2. 크로스 엔트로피 손실의 실제 코드**: 각 위치에서 `-probs[target_id].log()` 을 계산하고 평균을 내는 과정
* **6-3. Adam 옵티마이저 상세**: 1차 모멘트(m), 2차 모멘트(v), 편향 보정(bias correction)의 직관적 이해
* **6-4. 학습률 스케줄링과 그래디언트 초기화**: 선형 감쇠(`lr_t`)와 매 스텝 `p.grad = 0` 초기화의 필요성
