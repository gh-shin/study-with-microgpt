# 파이썬 기초부터 Autograd Engine 마스터 학습 플랜

`microgpt.py`의 `Value` 클래스(스칼라 기반 자동 미분 엔진)를 완벽히 이해하고 직접 구현할 수 있는 수준에 도달하기 위한 4단계 학습 플랜입니다.

## Phase 1: 파이썬 객체지향 및 고급 문법 마스터하기
계산 그래프를 코드로 표현하고, 수학 연산 기호를 파이썬 객체에 적용하기 위해 필요한 단계입니다.

1. **파이썬 기초 문법**: 변수, 조건문, 반복문, 리스트/튜플 다루기
2. **객체지향 프로그래밍 (OOP)**: 클래스와 객체의 개념, 인스턴스 변수
3. **파이썬 매직 매서드 (Dunder Methods) 심화**: 연산자 오버로딩, 우항 연산자 처리
4. **고급 파이썬 기술**: 재귀 호출(Recursion), 클래스 최적화(`__slots__`)
5. **고급 파이썬 구문**: 리스트 내포, lambda, 제너레이터 표현식, dict와 set

## Phase 2: 미분의 본질과 연쇄 법칙 (수학 기초)
머신러닝과 역전파의 톱니바퀴를 돌리는 수학적 기초를 쌓는 단계입니다.

1. **기초 함수와 미분법 (Calculus)**: 다항식, 자연로그, 지수 함수의 미분
2. **편미분 (Partial Derivative)**: 결과값에 여러 변수가 관여할 때의 미분법 (덧셈, 곱셈의 편미분)
3. **연쇄 법칙 (Chain Rule)**: 합성 함수를 미분할 때 바깥쪽부터 안쪽으로 미분값을 곱해 들어가는 원리
4. **확률 분포와 손실 함수**: 소프트맥스 변환과 크로스 엔트로피 손실, 수치 안정성 수식

## Phase 3: 딥러닝 코어 이론 - 계산 그래프와 역전파
배웠던 수학 연산을 알고리즘 자료구조의 형태로 컴퓨터에 옮기는 과정입니다.

1. **계산 그래프 (Computation Graph)**: 수식을 방향성 비순환 그래프(DAG) 형태의 자료구조로 변환
2. **순전파 (Forward Pass)**: 실제 값을 대입하여 계산을 수행해나가며 최종 결과를 구하는 과정
3. **위상 정렬 (Topological Sorting)**: 의존성이 있는 작업들의 순서를 정하는 알고리즘
4. **역전파 (Backward Pass) 알고리즘**: 위상 정렬 역순으로 체인 룰을 적용하여 미분값을 전파
5. **활성화 함수**: 비선형성의 도입과 ReLU 함수의 이해
6. **토큰화와 임베딩**: 문자를 정수 ID로(Token), 정수를 실수 벡터로(Embedding) 변환
7. **어텐션과 트랜스포머**: Query/Key/Value를 통한 중요도 가중합과 병렬 처리
8. **안정화 기법**: RMSNorm 정규화와 잔차 연결(Residual Connection)

## Phase 4: Autograd Engine 실전 구현 및 분석 (최종 목표)
지금까지 배운 것을 결합하여 현재 파일의 `Value` 클래스를 직접 구현해보는 단계입니다.

1. **단일 스칼라 노드 객체화**: 스칼라 값을 감싸는 `Value` 클래스 뼈대 작성
2. **연산 노드 생성기 구현**: `__add__`, `__mul__` 등 연산 수행 시 자식 노드를 연결하고 로컬 그래디언트를 정의
3. **DFS를 활용한 체인룰 계산기**: `backward()` 통합 메서드로 위상 정렬과 역전파 구현
4. **파라미터 초기화**: 모델 하이퍼파라미터 체계와 가우시안 랜덤 행렬 구조(`state_dict`)
5. **학습 루프와 옵티마이저**: 순전파/역전파 주기 반복 및 Adam 옵티마이저의 동작 원리
