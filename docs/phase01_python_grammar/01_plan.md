# Phase 1: 파이썬 객체지향 및 고급 문법 마스터하기

이 단계에서는 계산 그래프를 코드로 표현하고, 수학 연산 기호를 파이썬 객체에 적용하기 위해 필요한 핵심 파이썬 문법을 배웁니다. `microgpt.py`의 `Value` 클래스를 이해하기 위한 필수적인 파이썬 기능들을 다룹니다.

## 📌 Chapter 1. 파이썬과 데이터 다루기 기초
연산의 가장 기본 단위가 될 변수와 상태의 흐름을 통제하는 방식을 학습합니다.
* **1-1. 변수와 스칼라 값**: 숫자(정수와 실수)의 할당, 변수의 의미
* **1-2. 기본 자료구조**: 여러 개의 데이터를 보관하는 리스트(List)와 튜플(Tuple), 데이터의 순서(`index`)
* **1-3. 제어문과 함수**: 
  * `if`, `for`, `while` 문을 이용한 흐름 제어
  * `def` 예약어를 이용한 기본 함수 정의 방법 및 반환값(`return`)의 이해

## 📌 Chapter 2. 객체지향 프로그래밍 (OOP) 입문
단순한 형태의 숫자 자료형을 넘어, 값(data)과 기울기(grad), 계보(children)를 한 번에 저장할 수 있게 해주는 파이썬 클래스의 원리를 배웁니다.
* **2-1. 클래스(Class)와 객체(Object)**: 청사진과 실체의 개념 비교
* **2-2. 생성자와 상태 초기화**: 
  * `__init__` 메서드의 역할과 구동 원리
  * `self`란 무엇인가? 인스턴스 변수(`self.data`, `self.grad`)를 통한 상태 저장
* **2-3. 인스턴스 메서드(Instance Methods)**: 객체의 데이터에 접근하고 변경하는 함수 만들기

## 📌 Chapter 3. 매직 메서드 (Dunder Methods)와 연산자 오버로딩
`a + b`, `a * b` 와 같은 흔한 사칙연산 기호를 우리가 직접 만든 커스텀 객체(Value)에 적용하는 파이썬의 강력한 기능을 마스터합니다.
* **3-1. 생소한 밑줄 두 개(`__`)**: 매직 메서드의 개념과 객체 문자열 표현(`__repr__`)
* **3-2. 사칙연산 오버로딩 1 (기본 연산)**: 
  * 더하기: `__add__(self, other)`
  * 곱하기: `__mul__(self, other)`
  * 빼기 및 나누기: `__sub__`, `__truediv__`
* **3-3. 단항 연산자와 거듭제곱**: 부호 뒤집기(`__neg__`), 제곱 연산(`__pow__`) 구현하기

## 📌 Chapter 4. 매직 메서드 심화 및 타입 대응
우리가 만든 객체와 단순한 숫자(`float`, `int`)를 서로 섞어서 연산할 때 파이썬 에러가 나지 않게 하는 우회 및 대응 기법을 배웁니다.
* **4-1. 타입에 따른 분기 처리**: `isinstance(other, Value)`를 활용해 들어온 값이 숫자인지 객체인지 판단하고 감싸기(Wrap)
* **4-2. 우항 연산자(Right-side operations) 오버로딩**: 
  * 객체가 오른쪽 연산자로 올 때(`2.0 + a`) 대응하는 `__radd__`, `__rmul__` 
  * 순서의 중요성과 `__rsub__`, `__rtruediv__` 구현의 차이점

## 📌 Chapter 5. 고급 파이썬 활용 기술
코드의 효율성을 극대화하고, 이후 복잡한 그래프(Tree) 탐색 시 필요한 핵심 논리 구조를 훈련합니다.
* **5-1. 파이썬 메모리 최적화와 `__slots__`**: 
  * 기본 파이썬 객체가 `__dict__` 구조를 가져서 발생하는 메모리 낭비 문제
  * 수만 개의 다발로 생성될 노드(Node) 객체의 크기를 가볍게 해주는 `__slots__` 튜플 사용법 (코드 32번 줄 이해)
* **5-2. 재귀 호출 (Recursion)**: 
  * 특정 함수가 자신의 내부에서 자기 자신을 다시 호출하는 원리
  * 콜 스택(Call Stack)의 이해 및 무한 루프에 빠지지 않기 위한 탈출 조건
  * 역전파 시퀀스를 결정하는 `build_topo` 함수와 DFS(깊이 우선 탐색) 이해를 위한 핵심

## 📌 Chapter 6. 리스트 내포와 고급 파이썬 구문
`microgpt.py` 전체에 걸쳐 빈번하게 사용되는 파이썬 고급 표현식을 마스터합니다. 타 언어 개발자에게 특히 생소할 수 있는 구문들입니다.
* **6-1. 리스트 내포 (List Comprehension)**: `[표현식 for 변수 in 순회대상]` 패턴과 중첩 리스트 내포
* **6-2. Lambda 함수 (익명 함수)**: `lambda` 예약어를 이용한 한 줄짜리 함수 정의 문법. 파라미터 행렬 생성기(`matrix`) 이해의 핵심
* **6-3. 제너레이터 표현식과 내장 함수**: `sum()`, `max()`, `zip()` 내부에 직접 넣는 표현식과 그 효율성
* **6-4. 딕셔너리(Dict)와 f-string**: 키-값 쌍으로 모델의 가중치 저장소(`state_dict`)를 구성하는 패턴과 동적 키 생성
* **6-5. 집합(Set)과 `in` 연산의 성능**: 리스트와 집합의 탐색 속도 차이, `visited = set()` 패턴이 DFS에 쓰이는 이유
